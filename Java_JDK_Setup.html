Steps to Install JDK on Windows

Follow these simple steps to install JDK on your Windows machine:

  1. Download the JDK:
    • Go to the Oracle JDK download page.
    • Select the latest JDK version for Windows and download the .exe installer.
  2. Run the Installer:
    • Open the downloaded .exe file.
    • Follow the installation prompts, choosing default settings unless you need to change the installation path.
  3. Set Environment Variables:
    • Right-click on "This PC" or "My Computer" → Select "Properties".
    • Click "Advanced system settings" → "Environment Variables".
    • Under "System Variables", find "Path", click "Edit", and add a new entry with the path to the JDK bin folder (e.g., C:\Program Files\Java\jdk-xx\bin).
  4. Verify Installation:
    • Open Command Prompt and type: java -version .
    • If Java is installed correctly, youâ€™ll see the installed version.

Steps to Install JDK on Linux

Follow these simple steps to install JDK on your Linux machine:

  1. Open Terminal.
  2. Update Package Manager:
    • Run the following command to update the package list: sudo apt update .
  3. Install JDK:
    • Install OpenJDK by running: sudo apt install openjdk-17-jdk .
    • You can replace 17 with the desired version.
  4. Set Default Java Version (if multiple versions are installed):
    • sudo update-alternatives --config java .
    • Choose the correct version from the list.
  5. Verify Installation:
    • In the terminal, run java -version .

Steps to Install JDK on macOS

Follow these simple steps to install JDK on your macOS machine:

  1. Download the JDK:
    • Visit the Oracle JDK download page.
    • Download the latest JDK .dmg installer for macOS.
  2. Install the JDK:
    • Open the downloaded .dmg file.
    • Follow the instructions in the installer, keeping default settings.
  3. Set JAVA_HOME Environment Variable (optional):
    • Open Terminal and enter: export JAVA_HOME=$(/usr/libexec/java_home)
    • Add this line to your .bash_profile or .zshrc for it to persist across sessions.
  4. Verify Installation:
    • In Terminal, type: java -version .
    • You should see the version of Java installed.
