What is a Class?

A class in Java is a template or blueprint used to create objects. It groups related attributes (variables) and methods (functions) that define the behavior and characteristics of an object. By using classes, you can model real-world entities in a program.

Key Features of Classes:

  1. Encapsulation: Classes bundle data and methods together, protecting the internal state of objects.
  2. Reusability: Once a class is defined, it can be used to create multiple objects.
  3. Inheritance: Classes can inherit properties and methods from other classes.
  4. Polymorphism: A class can define methods that behave differently based on the object.

Syntax for Creating a Class

Here is the general structure of a class in Java:

class ClassName {

dataType attributeName;
      
  // Constructor
  ClassName(parameters) {
  // Initialization code
}
      
  // Methods
  returnType methodName(parameters) {
  // Method body
  }
}

Explanation of Syntax:

  1. class ClassName: The keyword class is followed by the name of the class. The name should follow Java naming conventions, starting with a capital letter.
  2. Attributes (Fields): Variables that hold data related to the object. Each attribute has a data type (e.g., int, String).
  3. Constructor: A special method used to initialize objects of the class. It usually sets the initial values of the attributes.
  4. Methods: Functions inside the class that define behaviors. Methods can return values or perform actions using the class's attributes.

Attributes and Methods in Detail


Attributes (Fields):

Attributes store the state or properties of an object. For example, in a Car class, attributes might include the color, model, and year. 

class Car {

String model;
int year;

Constructors in Java

A constructor is a special type of method used to initialize objects. It has the same name as the class and does not return any value. Constructors help set up initial values for object attributes.

Constructor Example: 
class Car {
    String color;
    String model;
    int year;
      
    // Constructor to initialize attributes
    Car() {
    System.out.println("This is a constructor");
  }
}

Car(): This constructor is parameterless and executes the print statement inside it.

Creating Objects

Once a class is defined, you can create objects (instances of the class). Objects are created using the new keyword, followed by the constructor of the class.

Example: 
public class Main {
    public static void main(String[] args) {
        // Creating an object of the Car class
        Car myCar = new Car();
        
        // Calling the method to display car details
        myCar.displayDetails();
    }
}
  1. Car myCar = new Car();: Creates an instance of the Car class named myCar and call's the constructor.
  2. myCar.displayDetails();: Calls the displayDetails() method to print the carâ€™s details.
