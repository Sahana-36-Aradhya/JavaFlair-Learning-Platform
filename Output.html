Let’s Talk About Printing in Java

At this point, you’re already on your journey into Java, so let's take a closer look at how to display output on the screen. Printing messages or information is one of the most basic tasks you’ll do, but it’s also essential for tracking how your code behaves.

Java provides three handy ways to print things to the screen, and here they are:

  1. System.out.println() - Prints with a new line at the end.
  2. System.out.print() - Prints without moving to a new line.
  3. System.out.printf() - Prints formatted output.

Let's Print Something with System.out.println()

Imagine you’re sending a message and hitting "Enter" after every sentence. That's what System.out.println() does—it prints your message and moves to the next line automatically. It's perfect when you want to keep things neat and organized. Ready to give it a try?

Try it yourself!

  • System.out.println("Learning Java is fun!");
You’ll get: 
  • Learning Java is fun!
The cursor moves to the next line, waiting for the next print command.

Please click on the 'Run' button to view the output.


Stay on the Same Line with System.out.print()

The System.out.print() method works similarly to System.out.println(), but it doesn't move to a new line after printing. This is handy when you want multiple pieces of information on the same line.

Try this and see:

  • System.out.print("Java ");
System.out.print("is awesome!");

You'll see this output on the same line:

  • Java is awesome!

Please click on the 'Run' button to view the output.


Format Your Output with System.out.printf()

The System.out.printf() method allows you to create formatted output. You can insert variables or numbers into your text in a neat way. Don't worry about all the details now; just give it a try:

Printing a Integer Value

An integer (int) is a data type used to represent whole numbers. You can display an integer value using System.out.printf(), which allows you to format the output neatly.

Try this and see:

  • System.out.printf("You have %d new messages", 5);

You have 5 new messages

  • You have 5 new messages

Please click on the 'Run' button to view the output.


Printing a Float Value

A float is a data type used to represent decimal numbers. When you want to display a number with a decimal point in a specific format, you can use System.out.printf().

Try this and see:

  • System.out.printf("Your average score is %.2f", 85.6789);

The output will be:

  • Your average score is 85.68

In this example, %.2f tells Java to display the float value with two decimal places.

Please click on the 'Run' button to view the output.


Printing a String

A string is a sequence of characters used to represent text. You can easily include text in your output using System.out.printf().

Try this and see:

  • System.out.printf("Welcome, %s!", "John");

The output will be:

  • Welcome, John!

In this example, %s is a placeholder for a string, which gets replaced by "John" in the output.

Please click on the 'Run' button to view the output.
