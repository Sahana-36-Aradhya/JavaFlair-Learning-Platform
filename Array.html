Arrays

Arrays allow you to store multiple values of the same type in a single variable. Instead of using multiple variables to store individual values, arrays help you handle large amounts of related data efficiently.

General Syntax:

dataType[] arrayName;
arrayName = new dataType[size];
  • dataType: Specifies the type of elements (e.g., int, String, double) that the array will hold.
  • arrayName: The name of the array variable, following standard Java naming conventions.
  • []: Indicates that the variable will store an array, not a single value.
  • new: Used to create a new instance of the array, allocating memory for it.
  • size: An integer that defines the fixed number of elements the array can hold, which cannot be changed after creation.

Accessing Array Elements

To access an individual element in an array, you refer to its index. Remember, array indexing in Java starts at 0.

System.out.println(fruits[0]);
System.out.println(numbers[4]);
  • Element Access: fruits[0] retrieves the first element "Apple" from the fruits array, and numbers[4] retrieves the fifth element 25 from the numbers array.
  • Output: The System.out.println statements print "Apple" and 25 on separate lines in the console.

Modifying Array Elements

You can change the value of any element in an array by directly assigning a new value to its index.

fruits[1] = "Mango";  
System.out.println(fruits[1]);
  • Element Modification: fruits[1] = "Mango"; updates the second element of the fruits array, changing it from "Banana" to "Mango".
  • Output: The System.out.println(fruits[1]); statement prints the updated second element, which is now "Mango", to the console.

Syntax

arrayName.length
  • arrayName: This represents the name of the array whose length you want to access. In this context, it is fruits, which is an array of strings.
  • .length: This property is used to get the total number of elements in the array. It is an attribute that provides the size of the array, allowing you to determine how many items it contains without needing to count them manually.

Array Length

The length property allows you to find out how many elements an array contains. This is especially useful for loops and operations that need to process every element in the array.

System.out.println(fruits.length);
  • Array Length Access: The expression fruits.length retrieves the number of elements in the fruits array. In this case, it will return 3, as there are three elements: "Apple", "Banana", and "Mango".
  • Output: The System.out.println(fruits.length); statement prints the length of the fruits array to the console, so executing this code will display 3.

Multi-Dimensional Arrays

dataType[][] arrayName = {
{element1, element2, ...},
{element3, element4, ...},
        ...
};
  • Array Declaration and Initialization:The code int[][] matrix = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}}; declares a two-dimensional array named matrix with three rows, each represented by inner braces {}.
  • Accessing an Element:The statement System.out.println(matrix[1][2]); retrieves the value at the second row and third column (which is 6) and prints it to the console.

Java also allows you to create arrays with more than one dimension. A two-dimensional array is often used for storing data in rows and columns, like a table:

int[][] matrix = {
    {1, 2, 3}, 
    {4, 5, 6}, 
    {7, 8, 9}
};
System.out.println(matrix[1][2]);
  • Two-Dimensional Array Declaration: The code int[][] matrix = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}}; declares a two-dimensional integer array named matrix and initializes it with three rows and three columns, where each inner brace {} represents a row of values.
  • Indexing: In the expression matrix[1][2], the first index 1 refers to the second row (since indexing starts at 0), and the second index 2 refers to the third column of that row, which contains the value 6.
  • Output: The statement System.out.println(matrix[1][2]); retrieves the value 6 from the specified position in the array and prints it to the console.

Summary

  • Arrays store multiple values in a single variable, making data management more efficient.
  • Array Elements are accessed and modified using an index, with the first index being 0.
  • The length property is used to determine the number of elements in an array.
  • Loops, such as for or for-each, are commonly used to iterate over arrays and access each element.
  • Multi-dimensional arrays allow you to create tables or grids of data.
