Understanding Java Break and Continue Statements

The break and continue statements control the flow of loops and switch blocks. The break statement exits a loop or switch immediately, transferring control to the next statement outside. Itâ€™s useful for stopping further execution when a condition is met. On the other hand, continue skips the current iteration of a loop and moves to the next one, allowing the loop to continue running. These statements offer flexibility, making loops and switches more efficient and responsive to conditions.

Break Statement

The break statement is used to exit a loop or switch case immediately, skipping any remaining iterations or cases.

Syntax of break Statement

break;

Example: Breaking Out of a Loop

for (int i = 1; i <= 10; i++) { 
  if (i == 5) {
      break;
    }
  System.out.println(i);
}
  • The loop prints numbers from 1 to 4.
  • When i reaches 5, the break statement terminates the loop, and Loop terminated. is printed.

Continue Statement

The continue statement skips the current iteration and moves to the next one, allowing you to bypass certain conditions.

Syntax of continue Statement

continue;
for (int i = 1; i <= 10; i++) {
  if (i % 2 == 0) {
      continue;
    }
  System.out.println(i);
}
  • The loop prints only odd numbers from 1 to 10.
  • When i is even, the continue statement skips the current iteration, and the loop proceeds to the next iteration.

Summary

  • break Statement:Use break to immediately exit a loop or switch statement, which is useful for stopping execution when a specific condition is met.
  • continue Statement:Use continue to skip the current iteration of a loop and proceed to the next iteration, helpful for bypassing certain conditions without terminating the loop.
  • Effective Control:Both break and continue enhance your control over loop execution, improving code efficiency and readability.
