Running the Java Program

< Previous Next >

After writing and saving the file, follow the following steps to run your code successfully:

  1. Opening the Command Prompt or Terminal:
    • On Windows
      • Press Windows + R to open the Run dialog box.
      • Type cmd and press Enter. This will open the Command Prompt.
    • On macOS/Linux:
      • Open the Terminal by searching for it in Spotlight (macOS) or in the Applications menu (Linux).
  2. Navigating to the File’s Location:
    • Use the cd (change directory) command to move to the folder where you saved your Java file.

      For example:

      • If your file is saved in a folder called JavaProjects on the Desktop, type : cd Desktop/JavaProjects
      • Press Enter to navigate to that directory.
  3. Compiling and Running Your Program:
    • To compile your program, type: javac HelloWorld.java
    • If no errors appear, your program has successfully compiled. Now, run it with: java HelloWorld
    • You should see "Hello, World!" printed in the terminal.

Summary

  • Open Command Prompt (Windows) or Terminal (macOS/Linux).
  • Use cd to navigate to your file’s location.
  • Compile with javac HelloWorld.java.
  • Run the program using java HelloWorld.
  • "Hello, World!" will appear if successful.
