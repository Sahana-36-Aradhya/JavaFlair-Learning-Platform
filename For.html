For Loop

The for loop is ideal when you know beforehand how many iterations are needed. It consists of three parts: initialization, condition, and increment/decrement.

Syntax of For Loop

for (initialization; condition; increment/decrement) {
  // code block to be executed
}
  • initialization: This is executed once at the start, where you typically initialize a variable (e.g., int i = 0).
  • condition: This is checked before each iteration. If true, the loop runs; if false, the loop stops.
  • increment/decrement: After each iteration, this is executed to update the loop control variable.

Example: Printing Even Numbers from 1 to 10

public class ForLoopExample {
  public static void main(String[] args) {
    for (int i = 1; i <= 10; i++) {
      if (i % 2 == 0) {
        System.out.println(i);
      }
    }
  }
}
Explanation:
  • We initialize i to 1.
  • The loop runs as long as i is less than or equal to 10.
  • We check if the number is even (i % 2 == 0) and print it if true.
  • After each iteration, i is incremented by 1.

Java For-Each Loop

The for-each loop, also known as the enhanced for loop, is specifically designed to iterate over arrays or collections like ArrayList. It is more readable and removes the need for managing an index.

Syntax of For-Each Loop

for (type element : array/collection) {
  // code block to be executed
}
  • type: The data type of elements in the array or collection.
  • element: Represents each element in the array or collection during each iteration.

Example: Iterating Over an Array of Names

public class ForEachExample {
  public static void main(String[] args) {
    String[] names = {"Alice", "Bob", "Charlie"};
      for (String name : names) {
        System.out.println(name);
      }
    }
  }
  • We have an array of strings called names.
  • The for-each loop goes through each name in the array and prints it, without needing to manage any index.

Important Points to Remember

  1. For Loop: Use it when the number of iterations is known or when you need fine control over the loop, such as using an increment or decrement.

Summary

  • For Loop: A for loop runs for a known number of iterations, with control over initialization, condition, and increment/decrement. It is ideal for tasks requiring specific numbers of iterations.
  • For-Each Loop: The for-each loop simplifies iteration over arrays and collections, removing the need to manage index and providing a cleaner, more readable structure.
  • Best Practices: Use for when precise control of the loop is required; use for-each for clean, concise iteration over data structures like arrays and collections.
