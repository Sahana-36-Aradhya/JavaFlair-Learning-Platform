Comparing Values:

          int b = 10;

Combining Booleans with Logical Operators:

Booleans become more powerful when you combine them with logical operators like && (AND), || (OR), and ! (NOT). These operators allow you to work with multiple conditions. 

int age = 25;
boolean isEligible = (age > 18) && (age < 65);
System.out.println("Is the person eligible? " + isEligible);

In this case, both conditions (age > 18 and age < 65) need to be true for isEligible to be true. Here, since 25 fits within the range, the result is true.
